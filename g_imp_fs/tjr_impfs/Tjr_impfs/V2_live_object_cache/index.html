Module Tjr_impfs.V2_live_object_cache

An LRU cache of live objects; used for files and dirs.

This code is a bit naive - it blocks when going to the lower layer, whereas a cleverer version might allow read-only access to in-cache entries whilst going to lower layer. Whether this is actually a bottleneck in practice I don't know.
