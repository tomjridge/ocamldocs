Module type Small_intf.T

This interface allows to create timers (essentially, a mutable count and a mutable total time); timers have an associated "at_exit" function which is called when the process exits; each timer tracks the number of calls (eg to a particular function) and the total duration of all calls; this is what is printed at exit.

type t
val create : string -> t

The string argument is a "name" used to distinguish different timers
